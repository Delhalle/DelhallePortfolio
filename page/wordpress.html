Contexte du projet

Lors de la première année de BTS, tous les mercredis nous avions 4 heures d'AP. Un mercredi il nous a été demandé de réaliser un site avec wordpress sur le thème que l'ont souhaité, afin de découvrir ce que nous propose wordpress. Une présentation était à réaliser auprès des professeurs afin de montrer le site que nous avions faits.

Description

WordPress est un système de gestion de contenu open-source sous licence GPLv2, ce qui signifie que tout le monde peut utiliser ou modifier le logiciel WordPress gratuitement.
Il permet à une personne de créer facilement son site internet.
Wordpress propose de nombreux plugins afin de réaliser n'importe quel type de site.
Avec WordPress, vous pouvez créer des :
Sites Web d’entreprises
Boutiques eCommerce
Blogs
Portfolios
C.V.
Forums
Réseaux sociaux
Sites d’adhésion
...à peu près tout ce qu’on peut imaginer d’autre.
Beaucoup d'entreprise utilise wordpress.
