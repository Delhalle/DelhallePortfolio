Contexte du projet

Ce projet est une application en php avec le modèle MVC à réaliser, afin de créer des stages à partir de données enregistrées dans un fichier .csv ou via un formulaire de saisie.
Ce projet va nous permettre de se familiariser avec le modèle MVC et les formulaires. C'est une activité guidée.

Description

Sur l'application en cliquant sur le bouton "Ajout de stage à partir d’un fichier", il sera possible pour l'utilisateur de sélectionner un fichier csv, si le fichier passe un message s'affiche comme quoi tous les enregistrements sont lus et traités, sinon un message d'erreur s'affiche. Si le message d'erreur s'affiche, il est possible à partir d'une fenêtre pop-up de modifier le fichier. En cliquant sur non on retourne au formulaire précédent. Si c'est sur oui, il obtient les enregistrements erronés et peut les modifier.

Si l'utilisateur clique sur "Saisie d’un stage", un formulaire s'affiche, il demande :
-Le nom de l'entreprise
-La nature du stage
-La situation géographique
-le salaire mensuel
-Et des cases à cocher

Au clique sur "Génération d’un des stages", un bouton s'affiche pour générer le fichier CSV. En cliquant dessus un message affichant le nombre d'enregistrements s'affiche et le fichier est généré.

Synthèse

Ce projet a été à réaliser en équipe, il m'a permis de découvrir les formulaires et le modèle MVC. Ce projet était guidé et donc assez facile. Mais il devient difficiles quand il est a réalisé davantage en autonomie.
